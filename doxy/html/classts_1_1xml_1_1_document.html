  • All directories in TSPLUGINS_PATH environment variable.

  • All directories in PATH (UNIX) or Path (Windows) environment variable.

True on success, false on error.

◆ validate()

bool ts::xml::Document::validate ( const Document &  model) const

Validate the XML document.

This is a minimal mechanism, much less powerful than XML-Schema. But since we do not support schema, this is a cheap alternative.

Parameters
[in]modelThe model document. This document contains the structure of a valid document, with all possible elements and attributes. There is no type checking, no cardinality check. Comments and texts are ignored. The values of attributes are ignored.
Returns
True if this document matches model, false if it does not.
