const std::string &  prefix = "",

ReportInterface &  report = (*ts::NullReport::Instance()) 
)

Constructor.

The load order is the following:

  • Default system lookup using filename string.
  • If filename is a base name (no directory), search it into the same directory as the executable.
  • Same as previous with specified prefix in base name. In all cases, if the filename does not contain a suffix, the standard system suffix (.so or .dll) is added.
Parameters
[in]filenameShare library file name. Directory and suffix are optional.
[in]prefixPrefix to add to filename if the file is not found.
[in]permanentIf false (the default), the shared library is unloaded from the current process when this object is destroyed. If true, the shared library remains active.
[in,out]reportWhere to report errors.
