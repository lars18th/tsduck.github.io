◆ Out()

std::ostream& utest::Out ( )

This static method returns a reference to an output stream which can be used by unitary tests to log messages.

A unitary test typically does not display anything. It simply performs assertions. A complete set of unitary test suites reports successes or failures using CppUnit.

However, there are cases where the unitary test may want to issue trace, log or debug messages. Such messages should be sent to this output stream.

By default, these messages are discarded. However, when the option -d (debug) is specified on the command line of the unitary test driver, these messages are reported on the standard error stream.

Returns
A reference to an output stream used to report debug messages.
