Detailed Description

Representation of a logical_channel_number_descriptor.

This is a private descriptor, must be preceeded by the EACEM/EICTA PDS.

Constructor using a variable-length argument list.

Each entry is described by 2 arguments: service_id and @ lcn. All services are marked as visible by default. All arguments are int, not uint16_t, since integer literals are int by default. The end of the argument list must be marked by -1.
