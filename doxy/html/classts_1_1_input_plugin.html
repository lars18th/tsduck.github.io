Detailed Description

Input tsp plugin interface.

All shared libraries providing input capability shall return an object implementing this abstract interface.

◆ receive()

virtual size_t ts::InputPlugin::receive ( TSPacket *  buffer,
size_t  max_packets 
)
pure virtual

Packet reception interface.

The main application invokes receive() to get input packets. This method reads complete 188-byte TS packets in the buffer (never read partial packets).

Parameters
[out]bufferAddress of the buffer for incoming packets.
[in]max_packetsSize of buffer in number of packets.
Returns
The number of actually received packets (in the range 1 to max_packets). Returning zero means error or end of input.
