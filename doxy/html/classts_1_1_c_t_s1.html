Detailed Description

template<class CIPHER>
class ts::CTS1< CIPHER >

Cipher text Stealing (CTS) mode, alternative 1.

Several incompatible designs of CTS exist. This one implements the description in:

  • Bruce Schneier, Applied Cryptography (2nd, Ed.), pp 191, 195
  • RFC 2040, The RC5, RC5-CBC, RC5-CBC-Pad, and RC5-CTS Algorithms
  • "CBC ciphertext stealing" in http://en.wikipedia.org/wiki/Ciphertext_stealing

CTS can process a residue. The plain text and cipher text sizes must be greater than the block size of the underlying block cipher.

CIPHERA subclass of ts::BlockCipher, the underlying block cipher.
